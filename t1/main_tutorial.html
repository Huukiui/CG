General

To draw fractal you need to select tab “Services” on top of the page and then select tab “Fractal” on the new page. Then you cant enter parameters and start process!

To work with color models you need to select tab “Services” on top of the page and then select tab “Color models” on the new page. Then you can experiment and adjust color models to your parameters!

To work with moving objects you need to select tab “Services” on top of the page and then select tab “Moving objects” on the new page.  Then you can experiment with our tools for moving objects!
